# ZodiacMatch — Chinese Zodiac Compatibility Tool

Find your animal match. Pick two Chinese zodiac signs and discover compatibility scores, deep insights, and element dynamics. Free, instant, no sign-up.

**→ https://zodiacmatch.xyz**

## Features

- Pick any two of 12 Chinese zodiac signs
- Instant compatibility score (0–100)
- Element interaction analysis (Five Elements cycle)
- Detailed strengths and challenges for each pairing
- All 66 unique pairings evaluated
- No account required, no downloads

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Vercel (hosting)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.
